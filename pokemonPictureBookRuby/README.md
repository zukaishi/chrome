## pokemon picture book ruby
ひらがなしか読めない子供がポケモンずかんで名前を調べるために用意したchromeアドオン

## how to use
1. chrome://extensions 　chromeでアクセス

2. デベロッパーモードをONにする

3. パッケージ化されていない拡張機能を読み込む

4. clone してきた中の、chrome_package を選択

5. https://zukan.pokemon.co.jp/ chromeでアクセス

6. 見たいポケモンをクリック
